V. parahaemolyticus

Vibrio parahaemolyticus, an emerging zoonotic pathogen, is able to cause rapid cell death using a type III secretion system on it's first chromosome. Although the links to pathogenicity are not completely clear, what is clear is that this molecular weapon is a powerful tool against human cells. Focusing on how T3SS-1 is acivated, we aim to outline how T3SS-1 performs in Vibrio parahaemolyticus borne gastroenteritis.
